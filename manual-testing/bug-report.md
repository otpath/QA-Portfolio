# Relatório de Bug

## Identificação
- **ID:** BUG-001
- **Título:** Erro ao realizar login com credenciais válidas

## Ambiente
- Navegador: Google Chrome
- Sistema Operacional: Windows 10
- Ambiente: Homologação

## Pré-condições
- Usuário cadastrado no sistema
- Acesso à tela de login

## Passos para Reproduzir
1. Acessar a tela de login
2. Informar usuário válido
3. Informar senha válida
4. Clicar no botão "Entrar"

## Resultado Esperado
Usuário autenticado com sucesso e redirecionado para a área logada.

## Resultado Atual
Mensagem de erro exibida informando falha na autenticação.

## Severidade
Alta

## Prioridade
Alta

## Evidências
- Screenshot ou vídeo (quando aplicável)

## Observações
Impacta diretamente o fluxo principal do usuário.
